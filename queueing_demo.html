式（M/M/c の要点）
a = λ / μ
ρ = λ / (c μ)
P0 = 1 / [ Σ_{n=0}^{c-1} (a^n / n!) + (a^c / (c! (1-ρ))) ]
Pwait = ErlangC = (a^c / (c! (1-ρ))) · P0
Lq = Pwait · ρ / (1-ρ)
L = Lq + a
Wq = Lq / λ, W = Wq + 1/μ
